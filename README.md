# 名字 Gay 指数计算器 🌈

一个用来"科学"计算任何名字 Gay 指数的纯前端小工具。

**纯属娱乐，请勿当真。**

## 这是什么？

输入一个名字（中文、日文、英文、网名都行），程序会通过一套~~胡编乱造的~~复杂算法，给出一个 0-100% 的"Gay 指数"。

核心逻辑大概是这样：

1. 把名字标准化（中文转拼音、日文转罗马音、统一小写）
2. 提取字母、数字、特殊字符
3. 用一个看起来很唬人的"矩阵运算"生成结果
4. 根据结果随机选一条讽刺话术

算法是确定性的，同一个名字每次算出来的百分比一样，但话术是随机的。

## 功能

- **本地计算** - 纯前端，不收集任何数据
- **历史记录** - 用 localStorage 存储最近 10 条结果
- **分享链接** - 生成一个包含名字的 URL，别人点开会自动计算
- **动画效果** - 进度条、打字机、彩虹特效

## 技术栈

- HTML + Tailwind CSS（CDN 版）
- 原生 JavaScript
- [pinyin.js](https://github.com/hotoo/pinyin) - 中文转拼音
- [wanakana](https://github.com/WaniKani/WanaKana) - 日文处理

没有构建步骤，打开 `index.html` 就能跑。

## 部署

随便找个静态托管丢上去就行。GitHub Pages、Vercel、Cloudflare Pages、甚至直接双击打开 HTML 文件都可以。

## 安全

最近修复了一个存储型 XSS 漏洞 —— 历史记录渲染用的是 `innerHTML` 直接拼接用户输入。现在改成 DOM API + `textContent` 了。

如果你发现其他安全问题，欢迎提 Issue。
